# WhenIsTheMatch

WhenIsTheMatch is a football web application that helps fans follow teams and competitions through fixtures, live scores, standings, match statistics, reminders, broadcast information, and highlights.

The project focuses on providing essential match information in a fast and straightforward interface, without requiring an account or a native mobile application.

## Features

* Upcoming and recent matches
* Live scores and key match events
* Competition standings
* Team and competition schedules
* Broadcast information
* Match reminders
* Video highlights
* Localized dates and times

## Philosophy

WhenIsTheMatch aims to provide a lightweight alternative to larger football platforms by focusing on the information most fans look for before, during, and after a match.
