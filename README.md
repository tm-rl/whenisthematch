# WhenIsTheMatch - Football Fixtures & Live Scores

Welcome to the public repository for **[WhenIsTheMatch](https://whenisthematch.com)**, a clean, high-performance football web application designed to help fans track upcoming fixtures, standings, and match results.

---

## 🚀 Key Features

* **Comprehensive Match Fixtures & Standings:** Up-to-date tables and schedules for major global leagues (Premier League, La Liga, Champions League, etc.).
* **Live Scores & Match Events:** Match updates for goals, cards, and substitutions, regularly refreshed during active windows.
* **Broadcast Information:** Quick insights on match cards showing available TV stations and broadcasting channels where available.
* **Video Highlights:** Quick access to official post-match video highlights, no geo-restriction walls.
* **No Account Required:** Access localized dates, times, and match reminders instantly without signing up.

---

## 🧠 Project Philosophy

WhenIsTheMatch was built as a lightweight alternative to bloated sports platforms. Engineered with a modern stack optimized for Core Web Vitals and fast page loads, the platform delivers essential football schedules and results instantly. By prioritizing speed and core data, it provides a seamless user experience across mobile and desktop devices without requiring a native app download or mandatory user accounts.

---

## 🌐 Infrastructure & Hosting

To maintain maximum uptime and global availability during peak match hours, the platform uses a resilient infrastructure split:

* **Application Hosting:** Deployed on **[Railway](https://railway.com?referralCode=O3ThJB)** for reliable, automated container management and low-latency server-side execution.
* **CDN & Edge Network:** Routed through **[Cloudflare](https://www.cloudflare.com/)** to leverage global edge caching and smart routing, ensuring that cached pages load instantly worldwide while reducing server load.
